## 一、项目概述

本课题旨在构建一套**黄金（GOLD）新闻实时分析系统**，实现从多源新闻抓取、LLM/词典情感分析、五因子趋势强度评分（L1–L4），到网格交易策略适配及 MT5 模拟盘执行的完整链路，并与既有量化回测框架对接，验证新闻冲击下网格策略的防护效果。

**核心目标：**

1. 实时抓取黄金相关新闻（金十数据、东方财富等）  
2. 判断利多/利空方向及趋势强度等级（L1 弱 ~ L4 极强）  
3. 根据新闻等级动态调整网格间距、暂停反向挂单、部分减仓  
4. 通过 FxPro MT5 模拟盘验证执行逻辑  
5. 回测对比「无新闻适配」与「新闻冲击适配」策略表现  

**技术栈：** Python 3.10+、DeepSeek API、MetaTrader5、Parquet 数据归档、YAML 配置

### 3.3 Phase 2 — 趋势评分与网格策略

| 功能 | 实现 |
|------|------|
| 五因子权重 | 情感 0.35 / 可信度 0.15 / 新颖度 0.20 / 共识 0.15 / 量级 0.15 |
| 趋势等级 | L1（弱）~ L4（极强），与设计文档区间一致 |
| 持续性评估 | `duration_estimator.py`，按事件类型估计影响时长 |
| ATR 动态间距 | `spacing = base × atr_factor × trend_factor` |
| ATR 来源 | MT5 实时优先，History Data CSV 兜底 |

**网格动作规则（与设计文档对齐）：**

| 等级 | 评分区间 | 动作 | 间距因子 |
|------|---------|------|---------|
| L1 | [-30, 30] | 维持现有挂单 | 1.0 |
| L2 | ±30~60 | 暂停反向挂单，保留顺势 | 0.8 |
| L3 | ±60~85 | 暂停新增，反向仓减 50% | 0.6 |
| L4 | ±85~100 | 全部暂停，空仓观望 | 0.5 |
