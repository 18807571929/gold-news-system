# 回测报告命名规范



## 格式



```

{YYYYMMDDHHmm}_{文件名}

```



| 部分 | 含义 | 示例 |

|------|------|------|

| YYYY | 年 | 2026 |

| MM | 月 | 07 |

| DD | 日 | 11 |

| HH | 时（24h，北京时间） | 19 |

| mm | 分 | 58 |

| 文件名 | 固定后缀，见下表 | `trades_baseline.csv` |



**示例：**



| 完整文件名 | 含义 |

|------------|------|

| `202607111959_report.md` | 2026-07-11 19:59 发起的 Markdown 报告 |

| `202607112007_trades_news_adapter.csv` | 2026-07-11 20:07 发起的 News Adapter 交易日志 |

| `202607111959_回测报告.html` | 给老师看的浏览器版摘要 |



同一分钟内重复运行，时间部分追加后缀：`202607111958_02_trades_baseline.csv`



## 输出路径（扁平，无子目录）



```

E:\量化项目\05 Backtest\results\news-grid\{时间}_{文件名}

```



给老师查阅的汇报副本：



```

E:\gold-news-system\docs\汇报\{时间}_{文件名}

E:\gold-news-system\docs\汇报\{时间}_回测报告.html

```



## 标准文件名后缀



| 后缀 | 说明 |

|------|------|

| `report.md` | Markdown 回测报告 |

| `comparison.json` | A/B 对比 JSON |

| `trades_baseline.csv` | Baseline 交易日志 |

| `trades_news_adapter.csv` | News Adapter 交易日志 |

| `equity_baseline.csv` | Baseline 权益曲线 |

| `equity_news_adapter.csv` | News Adapter 权益曲线 |

| `news_shock_events.csv` | 新闻冲击事件 |

| `shock_actions.csv` | 冲击策略动作 |

| `回测报告.html` | 浏览器摘要（仅 docs/汇报） |
