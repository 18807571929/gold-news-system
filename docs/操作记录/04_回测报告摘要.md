### 历史新闻数据

- 脚本：`python scripts/build_historical_signals.py --years 3`
- 输出：`02 History Data/signals/historical_signals.parquet`
- 来源：NFP（每月首个周五）+ CPI（每月 13 日）+ FOMC 决议日，共 **96 条 / 3 年**
- 若经济日历 Parquet 不足 1 年，自动回退到内置宏观日程

### 命令

```powershell
# 1. 构建 3 年历史信号
python scripts/build_historical_signals.py --years 3

# 2. 正式 1 年回测（500 刀）
python scripts/run_news_grid_backtest.py --years 1 --initial 500

# 3. 正式 3 年回测
python scripts/run_news_grid_backtest.py --years 3 --initial 1000

# 发烟测试（勿写入论文）
python scripts/run_news_grid_backtest.py --smoke --initial 500
```
