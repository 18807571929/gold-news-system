## 六、常见问题

**Q：工位 pip 装不上 MetaTrader5？**  
A：不装也行。工位只跑 NLP+回测；`requirements.txt` 里 MT5 包仅家里需要。

**Q：工位没有 `E:\量化项目`？**  
A：只拷 `gold-news-system` + `02 History Data/data/XAUUSD_H1.csv` 即可回测；或改 config 路径。

**Q：L2 验证 exit code 2？**  
A：休市 `market_closed: true`，换开盘时段再跑，不是程序坏了。

**Q：密码/API Key 在工位怎么办？**  
A：工位 config 用 `enabled: false` 可不填 MT5 密码；DeepSeek 用环境变量 `DEEPSEEK_API_KEY`，勿把 config 拷到公网。
