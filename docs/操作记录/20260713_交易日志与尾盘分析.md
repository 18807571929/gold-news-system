## 五、尾盘黄金为何暴跌（归因分析）

结合今日快讯链与价格路径，**不是单一事件**，而是「地缘溢价回吐 + 宏观预期切换 + 技术面连锁」叠加：

### 1. 地缘叙事反转（主因之一）

- **白天/上午：** 霍尔木兹、美伊冲突 → 油价飙升叙事 → 市场曾交易「通胀再起 / 避险」。
- **傍晚：** 金十「决战 CPI」稿称：美伊局势 **6 月已阶段性降温**，**油价大跌**；市场预期整体 CPI 月率 **由正转负**。
- **含义：** 地缘 risk premium 被抽走；黄金同时失去「避险买」与「通胀对冲」两条支撑 → **多头平仓 + 追空**。

### 2. 实际价格行为：慢跌变急跌（22:00 后）

信号中的 bid 路径：

```
4076（早盘）→ 4055（下午）→ 4040（22:05）→ 4036（22:06）→ 4020 下方（22:17）
```

22:05–22:17 **约 15 分钟再跌 ~20 USD**，属于 **整数关口 4040/4020 连续破位** 后的程序化/跟风卖盘，快讯标题也呈「滚雪球」式更新（-1.7% → -2% → -2.5%）。

### 3. 美元与利率预期

- 尾盘另有「欧洲央行降息预期下调、全球已九次降息」类宏观稿（偏 **利率路径上修** → 利空黄金）。
- 与「能源暴跌 → CPI 转负」稿并存时，市场短期更交易 **实际利率不上行但避险退潮** 的组合，对 gold 仍偏空。

### 4. 国内盘情绪

- 上金所 T+D **晚盘/收盘 -1.45%**（885.6 元/克），与 COMEX 走势共振，强化「全天弱势」标签。

## 六、结论与建议

**市场面：** 7月13日尾盘暴跌 = **地缘缓和 + 油价回落 + CPI 前通缩预期 + 4040/4020 技术破位** 的共振；快讯从「霍尔木兹推升通胀」切换到「能源暴跌 CPI 或转负」，叙事反转是核心。

1. 停止 `main.py --mode loop`，撤销剩余 **8** 笔挂单。
