# L2 挂单验证清单

**目标：** 验证 L2 利多信号能删除反向（sell）挂单，保留顺势（buy）挂单。

**前置：** MT5 已登录 FxPro；市场开盘；GOLD 有报价。

## 步骤

### 1. 连接测试

```powershell
cd E:\gold-news-system
python src/mt5_bridge/connector.py
```

确认 `"connected": true`，bid > 0。

### 2. 干跑预览

```powershell
python scripts/verify_l2_execution.py --dry-run
```

应看到计划动作：`cancel_reverse_pending(bullish)`。

### 3. 实盘验证（自动挂测试单 + 执行）

```powershell
python scripts/verify_l2_execution.py --setup
```

或双击 `scripts/L2验证.bat`。

### 4. MT5 界面核对

| 检查项 | 期望 |
|--------|------|
| sell limit 挂单 | 减少或归零 |
| buy limit 挂单 | 保留 |
| 日志 `verification.passed` | `true` |

### 5. 保存证据

- 截图：MT5「交易 → 订单」验证前后
- 日志：`docs/操作记录/logs/l2_verify_latest.json`
- 运行：`python scripts/generate_ops_log.py` 更新摘要

## 预期 JSON 片段
