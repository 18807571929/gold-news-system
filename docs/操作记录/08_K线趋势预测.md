# K 线趋势预测（不交易）

## 功能

在不执行任何交易的前提下，输出**未来 N 根 K 线**（默认 4 根 H1）的趋势方向判断。

## 判断逻辑

| 输入 | 权重 | 说明 |
|------|------|------|
| 最新新闻趋势分 | 55% | `sentiment_cache` 中最新一条的 `trend_score.composite_score` |
| 价格动量 | 45% | 近 4 根涨跌 / ATR + 短均长均差 |

方向：`看涨` / `看跌` / `震荡`（综合分超过 ±12）

第 2~N 根 K 线置信度逐根衰减，方向向中性轻微回归。

## 运行

```powershell
cd E:\gold-news-system
python scripts/run_kline_trend_forecast.py --bars 4 --timeframe H1
```

参数：

| 参数 | 默认 | 说明 |
|------|------|------|
| `--bars` | 4 | 预测 K 线根数 |
| `--timeframe` | H1 | H1 / M15 |

## 输出（命名：时间_文件名）

```
docs/汇报/{YYYYMMDDHHmm}_K线趋势预测.html
docs/汇报/{YYYYMMDDHHmm}_K线趋势预测.md
docs/汇报/{YYYYMMDDHHmm}_K线趋势预测.json
```

## 代码

- `src/forecast/kline_trend.py` — `KlineTrendForecaster`
- `scripts/run_kline_trend_forecast.py` — CLI + 文档生成

本功能**独立**于 `GridExecutor` / 回测；仅生成趋势判断文档，不下单。
