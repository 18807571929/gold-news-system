# 第 1 章 绪论

> **说明：** 毕设正文初稿。文献引用可在定稿时按学校格式补全。

## 1.1 研究背景与意义

黄金（XAUUSD/GOLD）兼具商品属性与金融属性，其价格在宏观数据发布、央行政策预期及地缘风险事件驱动下，常在较短时间内出现显著波动。对于采用网格等区间交易策略的投资者而言，固定参数的对称网格在新闻冲击窗口内容易在错误方向持续叠层；在小资金、高杠杆条件下，还可能触发保证金强平。

传统做法或依赖人工盯盘暂停交易，或采用与新闻分析割裂的纯技术面网格，难以形成「新闻理解 → 趋势强度 → 网格行为调整」的闭环。随着大语言模型（LLM）与结构化宏观数据源（如 FRED）的普及，构建一套可解释、可回测、可对接 MetaTrader 5（MT5）模拟盘的智能系统具有明确的工程价值与教学意义。

本课题面向**珠海科技学院智能系统方向**毕设要求，实现黄金新闻实时利多/利空分析与趋势强度判断，并在此基础上提出**新闻冲击网格适配策略（News Shock Grid Adapter）**，验证其在极端回撤与强平风险上的缓释效果。

---

## 1.2 国内外研究现状

**（1）金融文本情感分析。** 词典法、机器学习及 LLM 被广泛用于新闻、社交媒体的情绪抽取；黄金领域需结合宏观术语与品种特性构建专用词典或使用 Chain-of-Thought 提示。

**（2）事件驱动与 macro surprise。** 实际公布值相对预期的偏离（surprise）被用于解释资产短期反应；CPI、非农（NFP）、FOMC 等事件对金价方向具有一定统计关联，但命中率通常不足以单独构成 alpha，更适合作为风控触发条件。

**（3）网格交易与保证金模型。** 网格通过在价格区间内挂限价单获取震荡收益，但在趋势性冲击下暴露于单边叠层风险。现有教学或商业全量回测引擎往往与特定底仓策略绑定，不利于在毕设边界内独立验证「新闻适配层」的贡献。

**（4）与本课题的关系。** 本文不追求复刻某一闭源全量引擎，而聚焦**自有策略层**：L1–L4 分级冲击规则、surprise→动量方向链、以及自研 GridShockSimulator 上的 A/B 对比。

---

## 1.3 研究内容与主要工作

本课题主要工作包括：

1. **多源新闻与宏观数据管道**：金十/东财抓取、宏观日历与 FRED actual/forecast 合并、Parquet 归档。  
2. **情感与趋势评分**：词典/DeepSeek NLP、五因子 composite、L1–L4 等级输出。  
3. **新闻冲击网格适配策略**：间距因子、暂停反向挂单、L3 停新开、方向性网格等（详见第 4 章）。  
4. **自研回测引擎**：GridShockSimulator，支持 H1/M1、点差与保证金强平（详见第 5 章）。  
5. **MT5 模拟盘执行验证**：FxPro 模拟盘连接、L2 删反向挂单验证（详见第 6 章）。

---

## 1.4 主要创新点

1. 宏观 **actual vs forecast surprise** 与 H1 动量融合的方向解析链，及 CPI/NFP/PCE 等品类阈值分化。  
2. **价格动量与新闻方向冲突** 时的冲击降级（`neutral_widen_only` 保留 L3 停单保护）。  
3. 小账户 **100× 杠杆 + 保证金 stop_out** 场景下的 risk overlay 与 spacing/layers 参数扫描。  
4. 新闻信号 → ShockPolicy → MT5 挂单调整的闭环，与回测 timeline 语义对齐。
