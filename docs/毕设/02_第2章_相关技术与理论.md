# 第 2 章 相关技术与理论

---

## 2.1 网格交易原理

网格交易通过在价格区间内按固定间距布置限价买单与卖单，在震荡行情中反复「低买高卖」获取价差收益。设基准价为 \(P_0\)，间距为 \(\Delta\)，则买单价格序列为 \(P_0 - k\Delta\)（\(k=1,2,\ldots\)），卖单为 \(P_0 + k\Delta\)。当价格触及某层限价单时成交，并在相邻间距处挂出平仓或反向 replenishment 单。

**风险特征：** 在单边趋势或新闻冲击下，逆势侧连续成交会导致持仓与挂单同向累积，占用保证金上升。对于 **$500 初始资金、100× 杠杆** 的小账户，极端回撤可能触发 **stop_out**（本系统设定保证金比例 ≤ 50% 时强平）。

---

## 2.2 保证金与杠杆模型

本系统回测与 MT5 模拟盘均采用近似 CFD 保证金模型：

\[
\text{margin} = \frac{\text{price} \times \text{oz} \times \text{volume}}{\text{leverage}}
\]

其中 GOLD 默认 **0.01 手 = 1 盎司**。浮动盈亏按 mark-to-market 计算；可选 **全点差** `spread_price=0.20` USD：开多成交价 = 限价 + spread/2，平多按 bid 侧报价。

---

## 2.3 新闻情感分析与 LLM

**词典法：** 基于 `gold_sentiment.yaml` 对快讯标题进行 bullish/bearish/neutral 匹配，延迟低、可解释，适合实时 loop。

**LLM 法：** 使用 DeepSeek API 多跳 Chain-of-Thought，对复杂句式和隐含宏观含义进行解析，输出方向与置信度；需配置 API Key，成本高于词典。

**五因子趋势评分：** 情感幅度、共识、时效、持续性与价格上下文等加权得到 composite 分数，并映射为 L1–L4 等级，供 GridAdapter 与回测 ShockPolicy 共用。

---

## 2.4 宏观 Surprise 与金价

对含 `actual`、`forecast` 的宏观事件，定义意外程度：

\[
\text{diff} = \text{actual} - \text{reference}
\]

其中 reference 优先取 forecast，否则取 previous。经品类阈值检验后，将 diff 映射为 hawkish/dovish，再转换为对黄金的 bullish/bearish（例如 NFP 高于预期偏 hawkish → 利空黄金）。

**局限：** 部分 forecast 来自 FRED 朴素预期（如上期环比），非调查一致预期；surprise 含噪。本课题将其定位为 **冲击防护触发** 而非独立盈利信号（第 5 章命中率约 50%）。

---

## 2.5 H1 价格动量

当 surprise 不可用或列入排除品类（如 PCE 方向降权）时，取事件时点前 H1 收益率动量 \(m\)。当 \(|m| \ge 18\) 时作为方向兜底；与 surprise 冲突时保留 surprise 方向，仅下调 composite 分数。

---

## 2.6 MetaTrader 5 与 Python 桥接

MT5 提供官方 Python API（`MetaTrader5` 包），支持账户登录、品种报价、挂单/平仓与订单查询。本系统通过 `src/mt5_bridge/connector.py` 连接 FxPro 模拟盘，经 `GridExecutor` 将 ShockPolicy 转为删单、部分平仓与顺势布网格等动作。

本章介绍了网格交易、保证金模型、新闻情感分析、macro surprise、动量兜底及 MT5 桥接等理论与技术基础，为第 3 章系统设计与第 4 章策略规则提供概念支撑。
